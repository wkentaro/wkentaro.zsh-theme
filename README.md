# zsh-theme
Simple zshell theme for git user.

Installation
------------
```zsh
% git clone https://github.com/wkentaro/zsh-theme.git ~/.zsh-theme
% echo "source ~/.zsh-theme/theme.zsh" >> .zshrc
```
