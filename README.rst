Installation
============

.. code-block:: zsh

  % git clone https://github.com/wkentaro/wkentaro.zsh-theme.git ~/.wkentaro.zsh-theme
  % echo "source ~/.wkentaro.zsh-theme/wkentaro.zsh-theme" >> .zshrc
